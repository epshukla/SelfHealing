# Self-Healing Agent System

## Structure

- **src/agents/**: The "Brain" and tool capabilities.
- **src/engine/**: Reasoning, observing, and guardrails.
- **src/data/**: Vector store and knowledge base.
- **src/state/**: Persistent memory.
- **simulations/**: Generators and scenarios for testing.
- **ui/**: Dashboard and trace viewer.
- **tests/**: Verification tests.
